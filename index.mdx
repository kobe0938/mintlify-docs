Harbor is a Python framework that makes it easy to evaluate and optimize AI agents inside Docker containers. Whether you are benchmarking an off-the-shelf agent against Terminal-Bench, generating SFT traces for post-training, or running RL loops at scale with a cloud sandbox provider, Harbor gives you the modular building blocks to get there without reinventing the infrastructure.

## Get up and running in three steps

Follow these steps to install Harbor, run your first evaluation, and inspect the results.

<Steps>
  <Step title="Install Harbor">
    Install Harbor using `uv` (recommended) or `pip`. You need Python 3.10 or later and a running Docker daemon.

    ```bash
    uv tool install harbor
    ```
  </Step>
  <Step title="Run your first eval">
    Pick a registered dataset and point Harbor at your model and agent of choice. Harbor automatically downloads the dataset and spins up the container environment for each trial.

    ```bash
    harbor run -d terminal-bench/terminal-bench-2 -m "openai/o3" -a "terminus-2"
    ```
  </Step>
  <Step title="View results">
    Once the job finishes, inspect trial rewards and aggregate scores with a single command.

    ```bash
    harbor view jobs
    ```
  </Step>
</Steps>
